# i18n Arabic/English Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add Arabic/English language switching to the entire INFINDER frontend with RTL support, Cairo font for Arabic, and a navbar toggle.

**Architecture:** react-i18next with two typed TypeScript locale files (en.ts, ar.ts). Language stored in localStorage key `infinder_lang`. Switching Arabic sets `document.documentElement.dir = 'rtl'` and `lang = 'ar'`.

**Tech Stack:** react-i18next, i18next, i18next-browser-languagedetector, Cairo Google Font, Tailwind dir=rtl

```ts
import i18n from 'i18next';
import { initReactI18next } from 'react-i18next';
import LanguageDetector from 'i18next-browser-languagedetector';
import en from './locales/en';
import ar from './locales/ar';

i18n
  .use(LanguageDetector)
  .use(initReactI18next)
  .init({
    resources: {
      en: { translation: en },
      ar: { translation: ar },
    },
    fallbackLng: 'en',
    detection: {
      order: ['localStorage'],
      lookupLocalStorage: 'infinder_lang',
      caches: ['localStorage'],
    },
    interpolation: { escapeValue: false },
  });

function applyDir(lang: string) {
  document.documentElement.dir = lang === 'ar' ? 'rtl' : 'ltr';
  document.documentElement.lang = lang;
}

applyDir(i18n.language);
i18n.on('languageChanged', applyDir);

export default i18n;
```
